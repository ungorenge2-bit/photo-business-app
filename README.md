# photo-business-app

人物撮影向けの、撮影サイズ別・レンズ別の距離計算Webアプリ、燃費・ガソリン代計算アプリ、集合写真計算アプリです。iPhone Safariで開き、ホーム画面に追加して現場で使えます。

## アプリの使い分け

| 用途 | URL | iPhoneの名前 |
| --- | --- | --- |
| 撮影サイズ別の距離計算 | https://ungorenge2-bit.github.io/photo-business-app/ | 撮影距離 |
| 燃費・ガソリン代計算 | https://ungorenge2-bit.github.io/photo-business-app/fuel.html | 燃費計算 |
| 集合写真レイアウト・距離・ストロボ計算 | https://ungorenge2-bit.github.io/photo-business-app/group.html | 集合写真 |

- 撮影距離: 会議室やスタジオで、レンズと構図に必要な距離を確認する
- 燃費計算: 撮影現場までの走行距離から、交通費やガソリン代を確認する
- 集合写真: 人数、列数、レンズ、会場奥行き、ストロボ条件をまとめて確認する

## 集合写真アプリでできること

- 総人数、列数、前列人数を入力
- レンズ焦点距離と会場奥行きから必要距離を計算
- ISO、絞り、ストロボ灯数、光の使い方からストロボ目安を表示
- 列ごとの人数目安を表示
- 音声入力または文章入力で条件をまとめて反映

## 音声入力の例

```text
30人、3列、前列10人、35ミリ、奥行き10メートル
50人、5列、28ミリ、ISO400、F8、ストロボ2灯
40人、4列、前列10人、アンブレラ、ISO800、F11
20人、2列、前列10人、50ミリ、部屋8メートル
60人、6列、24ミリ、F11、ISO400、ソフトボックス、2灯
```

音声入力ボタンが使えないブラウザでは、入力欄をタップしてiPhoneキーボードのマイク入力を使い、そのあと `文章を解析` を押してください。

## 撮影距離アプリでできること

- 撮影サイズを選べる
- 全身、ニーショット、ミディアムショット、ミドルショット、バストアップ、アップ、顔アップに対応
- 身長と撮影サイズ比率から、写したい高さを計算
- 頭上余白と下側余白を入れて必要距離を計算
- 縦位置と横位置の両方を表示
- 24mm、35mm、50mm、85mm、135mm、200mmのレンズ比較
- 選んだレンズで撮影サイズ別に比較
- 比率、頭上余白、下側余白を手動調整
- iPhoneのホーム画面に追加して起動

## 燃費計算アプリでできること

- 走行距離からガソリン代を計算
- 車に表示される燃費 `km/L` で計算
- `L/100km` 表示にも対応
- ガソリン単価ごとの比較表示
- 台数を入れて合計費用を計算
- iPhoneのホーム画面に追加して起動

## 撮影距離の計算式

```text
写したい高さcm = 人物身長cm × 撮影サイズ比率
必要高さcm = 写したい高さcm + 頭上余白cm + 下側余白cm
必要高さmm = 必要高さcm × 10
撮影距離mm = 焦点距離mm × 必要高さmm ÷ センサー高さmm
撮影距離m = 撮影距離mm ÷ 1000
```

35mmフルサイズとして計算しています。

```text
縦位置: センサー高さ36mm
横位置: センサー高さ24mm
```

## 燃費計算の式

```text
必要なガソリン量 = 走行距離 ÷ 燃費
1台あたりの費用 = 必要なガソリン量 × ガソリン単価
合計費用 = 1台あたりの費用 × 台数
```

例:

```text
120km ÷ 12km/L = 10L
10L × 180円/L = 1,800円
```

## GitHub Pagesで公開する方法

1. GitHubでこのリポジトリを開く

   ```text
   https://github.com/ungorenge2-bit/photo-business-app
   ```

2. 上部メニューの `Settings` を開く

3. 左メニューの `Pages` を開く

4. `Build and deployment` の `Source` を `Deploy from a branch` にする

5. `Branch` を次のように設定する

   ```text
   main / root
   ```

6. `Save` を押す

7. 数十秒から数分待つ

8. 公開URLを開く

   ```text
   https://ungorenge2-bit.github.io/photo-business-app/
   ```

## iPhoneで使う方法

撮影距離アプリ:

```text
https://ungorenge2-bit.github.io/photo-business-app/
```

燃費計算アプリ:

```text
https://ungorenge2-bit.github.io/photo-business-app/fuel.html
```

集合写真アプリ:

```text
https://ungorenge2-bit.github.io/photo-business-app/group.html
```

1. iPhoneのSafariで使いたいURLを開く
2. 画面下の共有ボタンを押す
3. `ホーム画面に追加` を押す
4. 名前を `撮影距離`、`燃費計算`、`集合写真` のいずれかにする
5. ホーム画面のアイコンから起動する

## 現場での使い方

1. 撮影前の見積や移動費確認は `燃費計算`
2. 個人撮影でレンズと構図を確認する時は `撮影距離`
3. 集合写真の人数、列、会場奥行き、ストロボ条件を確認する時は `集合写真`
4. よく使うものはiPhoneのホーム画面に追加しておくと、Safariを探さずに起動できます

## 注意

バストアップ、ミディアム、ニーショットなどの範囲は、撮影現場やカメラマンによって多少解釈が変わります。このアプリでは、身長に対する比率で簡易計算しています。集合写真計算も、人物の体格、椅子、段差、背景、天井高、壁や天井の色、ストロボ出力によって結果が変わります。実際の撮影ではテスト撮影で確認してください。

`file:///tmp/...` のようなMac内の一時ファイルURLは、iPhoneでは開けません。iPhoneで使うには、GitHub Pagesなどで公開された `https://...` のURLをSafariで開いてください。
