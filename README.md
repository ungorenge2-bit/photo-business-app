# photo-business-app

人物撮影向けの、撮影サイズ別・レンズ別の距離計算Webアプリです。iPhone Safariで開き、ホーム画面に追加して現場で使えます。

## できること

- 撮影サイズを選べる
- 全身、ニーショット、ミディアムショット、ミドルショット、バストアップ、アップ、顔アップに対応
- 身長と撮影サイズ比率から、写したい高さを計算
- 頭上余白と下側余白を入れて必要距離を計算
- 縦位置と横位置の両方を表示
- 24mm、35mm、50mm、85mm、135mm、200mmのレンズ比較
- 選んだレンズで撮影サイズ別に比較
- 比率、頭上余白、下側余白を手動調整
- iPhoneのホーム画面に追加して起動

## 計算式

```text
写したい高さcm = 人物身長cm × 撮影サイズ比率
必要高さcm = 写したい高さcm + 頭上余白cm + 下側余白cm
必要高さmm = 必要高さcm × 10
撮影距離mm = 焦点距離mm × 必要高さmm ÷ センサー高さmm
撮影距離m = 撮影距離mm ÷ 1000
```

35mmフルサイズとして計算しています。

```text
縦位置: センサー高さ36mm
横位置: センサー高さ24mm
```

## 撮影サイズの初期比率

| 撮影サイズ | 画面に入れる高さの目安 | 頭上余白 | 下側余白 |
| --- | ---: | ---: | ---: |
| 全身 | 身長の100% | 10cm | 10cm |
| ニーショット | 身長の75% | 8cm | 8cm |
| ミディアムショット | 身長の60% | 7cm | 7cm |
| ミドルショット | 身長の50% | 6cm | 6cm |
| バストアップ | 身長の35% | 5cm | 5cm |
| アップ | 身長の25% | 4cm | 4cm |
| 顔アップ | 身長の18% | 3cm | 3cm |

## GitHub Pagesで公開する方法

1. GitHubでこのリポジトリを開く

   ```text
   https://github.com/ungorenge2-bit/photo-business-app
   ```

2. 上部メニューの `Settings` を開く

3. 左メニューの `Pages` を開く

4. `Build and deployment` の `Source` を `Deploy from a branch` にする

5. `Branch` を次のように設定する

   ```text
   main / root
   ```

6. `Save` を押す

7. 数十秒から数分待つ

8. 公開URLを開く

   ```text
   https://ungorenge2-bit.github.io/photo-business-app/
   ```

## iPhoneで使う方法

1. iPhoneのSafariで次のURLを開く

   ```text
   https://ungorenge2-bit.github.io/photo-business-app/
   ```

2. 画面下の共有ボタンを押す

3. `ホーム画面に追加` を押す

4. 名前を `撮影距離` のまま追加する

5. ホーム画面のアイコンから起動する

## 現場での使い方

1. 人物の身長を入力
2. 撮影サイズを選ぶ
3. `レンズ比較` で、24mmから200mmまでの必要距離を見る
4. `撮影サイズ比較` で、選んだレンズごとの構図差を見る
5. 部屋の奥行きを入れて、現場で足りるか判定する
6. 必要なら詳細設定で比率と余白を調整する

## 注意

バストアップ、ミディアム、ニーショットなどの範囲は、撮影現場やカメラマンによって多少解釈が変わります。このアプリでは、身長に対する比率で簡易計算しています。実際の撮影では、顔の大きさ、姿勢、髪型、衣装、手の位置、構図の余白によって必要距離が変わります。

`file:///tmp/...` のようなMac内の一時ファイルURLは、iPhoneでは開けません。iPhoneで使うには、GitHub Pagesなどで公開された `https://...` のURLをSafariで開いてください。
